# 🔍 Аналитический разбор

### 1. Help Mom-and-Pop Businesses Adopt AI

Консалтинг/услуга: приходишь к малому бизнесу (кафе, салон, мастерская) и внедряешь AI-инструменты под их процессы. Бизнес-модель — не продукт, а сервис.

- **Актуальность:** высокая — разрыв между возможностями AI и тем, что реально использует малый бизнес, огромный.
- **Стек:** технически лёгкая — no-code/low-code инструменты (Zapier, Make), готовые LLM (ChatGPT, Claude), CRM-интеграции. Кода почти не нужно.
- **Скиллы:** не программирование, а продажи, коммуникация, понимание бизнес-процессов, умение объяснять просто.
- **Конкуренция:** фрагментирована и локальна — конкурируешь за свой город/нишу, входной барьер низкий.

### 2. AI Tutoring and Personalized Education

AI-репетитор за $20/месяц вместо дорогого образования. Любой предмет, бесконечное терпение.

- **Актуальность:** высокая — самое прямое и понятное применение LLM.
- **Стек:** LLM API (OpenAI/Anthropic) с RAG по учебным материалам. Веб/мобильное приложение — Next.js или React Native. Knowledge tracing для адаптации (Python, `pyBKT`). Бэкенд FastAPI, PostgreSQL.
- **Скиллы:** промпт-инжиниринг, фуллстек-веб, RAG, педагогический дизайн.
- **Конкуренция:** очень плотная — выигрыш только через узкий предмет/аудиторию (напр. подготовка к конкретному экзамену).

### 3. Deploy Voice AI Agents for Local Businesses

AI-ресепшен для стоматологов, автосервисов — отвечает на звонки, записывает клиентов. $1K–$10K/месяц с клиента.

- **Актуальность:** высокая — голосовые AI-агенты стали реально юзабельны, у локального бизнеса понятная боль (пропущенные звонки).
- **Стек:** голосовые платформы — Vapi, Retell AI, Bland AI (готовая телефония + LLM). Логика — LLM API. Интеграции календарей — Google Calendar API, CRM. Минимум собственного кода.
- **Скиллы:** интеграция API, настройка готовых платформ, продажи локальному бизнесу.
- **Конкуренция:** растёт быстро, но рынок огромный и локальный — места хватает.

### 4. Build "Last Mile" AI Agents for Specific Industries

Глубокий вертикальный AI-агент под одну индустрию (стройка, право, фитнес). Маленькие команды строят крупные компании.

- **Актуальность:** очень высокая — главный тренд: побеждают узкоспециализированные агенты, а не универсальные.
- **Стек:** Python. Агентные фреймворки — LangGraph, CrewAI. RAG — LlamaIndex + векторная БД (Qdrant/Pinecone). Бэкенд FastAPI, фронт Next.js. Интеграции отраслевого софта.
- **Скиллы:** проектирование агентных пайплайнов, RAG, и главное — глубокое знание выбранной индустрии.
- **Конкуренция:** по каждой нише пока низкая — кто первый зашёл глубоко, тот и выиграл.

### 5. Convert Your Agency Into an AI SaaS Platform

Превратить ручную услугу агентства в self-serve AI-инструмент. Пример: 8 500 клиентов, без внешних инвестиций.

- **Актуальность:** высокая — переход от «продаём время» к «продаём продукт» масштабируется кратно.
- **Стек:** классический SaaS — Next.js фронт, Python/FastAPI или Node.js бэкенд, PostgreSQL, Stripe для оплат. LLM API для автоматизации ручной работы.
- **Скиллы:** фуллстек-веб, продуктовое мышление, биллинг/подписки. Нужна уже существующая агентская база.
- **Конкуренция:** зависит от ниши — преимущество в том, что у тебя уже есть клиенты и понимание процесса.

### 6. Vibe-Code a Domain-Specific App and Sell It

*(в исходном посте без описания)* По смыслу: быстро собрать узкоспециализированное приложение через AI-кодинг и продать.

- **Актуальность:** высокая — порог создания софта резко упал благодаря AI-инструментам.
- **Стек:** AI-кодинг — Cursor, Claude Code, v0. Обычно веб-приложение: Next.js + Supabase (быстрый бэкенд + БД), Stripe для оплат, Vercel для деплоя.
- **Скиллы:** базовый веб-стек, умение работать с AI-инструментами кодинга, выбор ниши и маркетинг.
- **Конкуренция:** высокая по числу таких же «vibe-кодеров» — выигрывает выбор ниши и дистрибуция, а не код.

### 7. Automate Boring Niche Vertical SaaS

Найти скучную индустрию, где много ручной агентской работы, и безжалостно автоматизировать. Узкая специализация — ключ.

- **Актуальность:** высокая — «скучные» рынки недоинвестированы в софт, конкуренции мало.
- **Стек:** SaaS-стек — Next.js, FastAPI/Node.js, PostgreSQL, Stripe. LLM API + RPA/автоматизация (Playwright) для замены ручных операций.
- **Скиллы:** фуллстек-веб, автоматизация процессов, и снова — глубокое знание скучной ниши.
- **Конкуренция:** низкая по определению (скучные рынки никому не интересны) — в этом и весь смысл.
